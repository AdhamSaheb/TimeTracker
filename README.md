# TimeTracker

This is an employee-task management system: 

Functionalities: 

* Admin Users can create, edit, delete other users 
* Manager Users can create, edit, delete projects 
* Manager Users can add tasks to a specific project, and give it to an employee user 
* Employee users can add time slots of work to each task given to them, add their own tasks to projects they√ïre in, and finish the tasks if done.

Technologies: 

* Backend: Django (REST Framework) 
* Frontend: Angular 10 

Supervision: 

* Exalt Technologies
